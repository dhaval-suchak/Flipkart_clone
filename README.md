📌 Project Overview

This project is an Android-based Flipkart Clone application developed to replicate the core functionalities of the Flipkart e-commerce platform. The application provides users with a smooth shopping experience including product browsing, product details, and user-friendly navigation.

The purpose of this project is to demonstrate Android development skills, UI design, and basic e-commerce functionality implementation.



🚀 Features

User-friendly Flipkart-like interface

Product listing and browsing

Product detail view

Category-based product display

Scrollable home page similar to e-commerce apps

Modern Android UI components

Responsive design for different screen sizes
